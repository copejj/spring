The Brain Dribbler project is an experiment in full-stack edge computing, running on the following specs:

Hardware & Operating System

  • Hosted on a Raspberry Pi - Model 4
  • Running Ubuntu Linux - v24.04 LTS

Web Server & Proxy

  • Reverse proxy from Nginx - v1.24.0
  • to Apache Tomcat - v10.1.34

DevOps & Language

  • Java - v21.0
  • GitHub Repository - Automated CI/CD
  • Google Gemini - AI-Assisted Development & Optimization

Programming Language & Core

  • The powerful and mighty Java - v21.0

Application Frameworks

  • Spring-Boot Framework - v3.4.2
  • Thymeleaf Templates - v3.1.3

Database

  • PostgreSQL Database - v16.11
